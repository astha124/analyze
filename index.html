Data Processing Project

Automated data conversion, processing, and reporting.

Project Overview

This project demonstrates a streamlined workflow for handling data, from an initial Excel file to a processed JSON output, all automated via GitHub Actions.

  • Data Source: data.xlsx (converted to data.csv)
  • Processing Script: execute.py (Python 3.11+, Pandas 2.3)
  • Output: result.json (generated in CI/CD)

The execute.py script processes the data.csv file, performs necessary data cleaning (e.g., handling non-numeric values and missing data), and aggregates information into a structured JSON format.

CI/CD and Results

A GitHub Actions workflow is configured to automatically:

  • Run code linting using Ruff.
  • Execute execute.py to generate result.json.
  • Publish result.json to GitHub Pages.
